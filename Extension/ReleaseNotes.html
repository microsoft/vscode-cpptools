Microsoft C/C++ Extension for VS Code

September 2018 Update

Thank you for installing the C/C++ extension. In the September update, we added a preview of semantic-aware Go To Definition. This allows Go To Definition to work for local variables. In addition, when the Go To Definition request references a symbol defined in the current translation unit, other incorrect symbols in the workspace with the same identifier will be filtered from the results.

We also added a new setting: C_Cpp.updateChannel to facilitate installing Insiders releases of the C++ extension.

Additional features and bug fixes are detailed in the full release notes.
